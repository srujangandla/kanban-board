# 📌 Kanban Board

A simple Trello-style Kanban Board built using HTML, CSS, and JavaScript. The application allows users to create task cards, drag and drop them between workflow columns, and automatically saves all tasks using the browser's localStorage so they remain available after refreshing the page.

---
## Live Demo

       https://kanban-board-umber-one.vercel.app

---
## 🚀 Features

* Create new task cards
* Drag and drop cards between columns
* Three workflow stages:

  * To Do
  * In Progress
  * Done
* Automatic data persistence using localStorage
* Responsive and clean user interface
* No external libraries or frameworks required

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Browser LocalStorage API

## ▶️ Getting Started

1. Clone the repository.

```
git clone https://github.com/srujangandla/kanban-board
```

2. Open the project folder.

3. Open `index.html` in your browser.

No installation or additional dependencies are required.

---

## 💡 How to Use

1. Enter a task in the input field.
2. Click **Add Card** to create a new task.
3. Drag a card to another column to update its status.
4. Refresh the page to see that all tasks are automatically restored.
